# Tabelas-Hash-Estrutura-de-Dados-
Projeto - Hashing e Colisões - Implementação de Tabelas Hash em Java com Endereçamento Aberto

Tabelas Hash (Hash Table) com Endereçamento Aberto

Este repositório contém a implementação desenvolvida de Estrutura de Dados, focada no funcionamento de Tabelas Hash utilizando endereçamento aberto (Open Addressing) para resolução de colisões.

Os códigos exemplificam:

Criação de tabela hash

Inserção de elementos

Função hash

Tratamento de colisões

Execução e testes

## Arquivos

HashFunction.java - 
Contém a função hash utilizada para mapear uma chave a um índice da tabela.

HashTable.java - 
Implementação base de uma tabela hash.

HashTableOA.java - 
Implementação da tabela hash usando Open Addressing (endereçamento aberto).

HashTableData.java - 
Classe auxiliar que armazena os dados dentro da tabela.

InsertResult.java - 
Representa o resultado da operação de inserção (sucesso, colisões, índice final etc.).

MainOA.java - 
Classe principal que executa os testes da tabela hash.


## Objetivo 

Demonstrar:

Funcionamento da função hash

Inserção com colisões

Aplicação de endereçamento aberto

Organização do código em múltiplas classes
